website : collection of webpage ..

webpage : webpage is an electronic document which has multimedia,hyper text content which runs over the internet 

-> we have 3 types 
	Static website : a website which has same content through out a time 
		ex: portfolios, college portals
	Dynamic website : a website which can change its content during regular intervals of time 	
	ex: cricbuzz, weather app..
	Web application : a dynamic website where the data is exchanged between two entities (system and user)
	ex: Instagram,facebook,gmail...

		SPA - Single Page apps (serverless or Servered)
		EE - Enterprise edition apps

HTML Basic elements : 

2 basic elements -> Tag, Attribute 

Tag : a unique keyword which enclosed in <>
-> the tag has two parts opening - <keyword>
			 closing - </keyword>

-> we have two types of tags -> paired tags/ container/ dependent
				unpaired tags/ empty/ independent 
	
-> paired tags : the tag which has both opening and closing part 
	ex: <h1></h1>,<p></p>
-> unpaired tags: the tag which has only opening part 
	ex: <img>, <br>,<input>,<hr>


Attribute : the properties of html tag 
-> key="value"
ex: <h1 align="center">Hello World</h1>
