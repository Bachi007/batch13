css selectors : selector is a keyword which is used to apply a block of css to a html element 

->types of selectors 
	tag name selector
	class name selector
	id name selector
	universal selector
	combinational selector
	

tag name selector: byusing tag name of a html element we can apply block of css to the element 

h1{
color:red;
}

class selector: we can select individual elements using its class name to apply a block of css, here we need to use a delimiter called .(dot) to represent class
-> class name can be duplicate 

id: we can select one unique element using it to apply a block of css, here id cannot be duplicate, we need to use a delimiter called #(hash) to represent id

universal : to apply a block of css to every element in the document, universal identified by * as delimiter



specificity : it is an algorithm used by browser to identify which css rule needs to apply for an element when multiple css rules are targeting same element
